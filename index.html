Pokemon JavaScript Catch

Instructions

Voici un exemple du code HTML d'un Pokémon :

<div class="pokemon type-plante type-poison" id="bulbizarre">
    <img src="./images/bulbizarre.png" alt="Bulbizarre">
    <h2 class="pokemon-name">Bulbizarre</h2>
    <p class="description">Type : Plante/Poison</p>
</div>

Objectifs de l'exercice

  1. Sélectionner Pikachu avec document.getElementById()
  2. Sélectionner Lokhlass avec document.querySelector() et changer sa couleur de fond en blanc
  3. Sélectionner tous les Pokémon de type eau avec document.querySelectorAll()
  4. Sélectionner tous les titres <h2> des Pokémon (pas de tout le document) et changer leur contenu par "Pika Pika"
  5. Supprimer tous les paragraphes <p> contenus dans les cartes Pokémon
  6. Ajoute Raichu qui est de type Électrique. Le nom de son image raichu.png.
    Les pokémons sont contenus dans : <div class="pokemon-container">
